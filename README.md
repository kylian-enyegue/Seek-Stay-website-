# Seek-Stay-website-
Sujet 1 : Gestion des logements pour étudiants
Il s’agit d’avoir un site Web qui permet à des étudiants en recherche de logement de
rechercher et d’entrer en contact avec un organisme ou un particulier. Le site devra ainsi
permettre l’ajout, la modification ou la suppression d’une nouvelle annonce par les
particuliers ou par les organismes tels que le CROUS. Les étudiants pourront en sélectionner
certains comme favoris, déposer une candidature ou simplement avoir des renseignements.
On peut également considérer les annonces faites par les étudiants pour trouver des
colocataires.
